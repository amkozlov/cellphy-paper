Scripts to convert tool outputs into standardized Newick tree format

- CELLPHY
  
  no conversion needed


- SIFIT

  $ ./sifit2tree.sh sifit.log.0001


- TNT

  ./tnt2tree.sh snv_hap.0001.TNT.Tree


- oncoNEM

  $ ./oncoNEM2tree.py snv_hap.0001.Edges snv_hap.0001.CellPops.txt


- infSCITE

 $ perl taxnameconvert.pl snv_haplotypes_dir/snv_hap.0001.missing.names snv_hap.0001_ml0.newick > snv_hap.0001_ml0.final-end.newick

  (snv_hap.*.names files generated by 02_convert_input/snv2infSCITE.py)


- SciPhi

!!! ETE3 required !!!
  
  $ ./sciphi2tree.sh vcf.0001.gv 0001


- ScisTree

  $ ./scis2tree.sh scis.log.0001


- SASC

!!! ETE3 required !!!

  $ ./sasc2tree.sh snv_hap.0001.missing_mlt.gv


- SPHYR

!!! ETE3 required !!!

  $ ./sasc2tree.sh snv_hap.0001.sphyr.dot
